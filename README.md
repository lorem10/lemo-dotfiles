## Installation

0. **Install dependencies.**

1. Clone this repository.

```sh
git clone https://github.com/lorem10/lemo-dotfiles.git

cd lemo-dotfiles/
```

2. Copy the all configs

```sh
cp -r config/* ~/.config/
```
